Programming lab 2: Testing similarity of regular expressions automaton

Submission deadline:

Introduction

The aim of this programming assigment is to write a Haskell program that given two regular expressions R, S determines if they are similar. 1

Assignment (80%)

  • Download from the GitHub repository cb0081-2014-2-lab2 the modules RegEx.hs and Example.hs, where you will find an implementation and some examples of a polyphormic regular expression data type.
  • (70%) Using the module RegEx, define a function 
      areSimilar :: RegEx a -> RegEx a -> Bool
    where areSimilar r s determines if regular expressions r and s are similar.
  • (10%) Annotate your source code in a way that explains your solution for the assignment. Improve it with HLint.

Examples

Using the module Example:

ghci> areSimilar universe1 universe2
True

ghci> areSimilar r s
True

ghci> areSimilar r t
False
